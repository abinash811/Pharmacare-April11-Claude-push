Unified Button System — ui/button.jsx (single source)

AppButton.jsx is deprecated. All button usage moves to <Button> from @/components/ui/button. Same 6 variants, same 4 sizes, one file.

outline Tertiary actions alongside a primary button. #fff border → #f9fafb
destructive Irreversible actions only. Never in bulk. #dc2626 → #b91c1c
ghost Icon buttons, table row actions, subtle links. transparent → #f3f4f6
link Inline text links, "skip" actions. transparent underline
